# Product Launch Template Changelog

All notable changes to the product launch template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this template adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-07

### Added
- Initial 9-question template structure
- YAML frontmatter for version tracking
- Gaps summary table
- Narrative threading section
- All 9 sections:
  1. Product Overview
  2. Target Audience
  3. Problems & Pain Points
  4. Competitive Landscape
  5. Differentiation
  6. Switching Dynamics
  7. Customer Language
  8. Proof Points
  9. Goals

### Template Authors
- Shahrukh Rao (initial design)
- Product Comms Agent (ongoing maintenance via weekly ratchet)

## Template Versioning Rules

- **MAJOR (x.0.0):** Adding/removing entire sections, changing target audience
- **MINOR (0.x.0):** Reordering sections, adding sub-fields, changing prompts
- **PATCH (0.0.x):** Typo fixes, clarification of existing prompts

## Governance

- Template changes require CMO approval
- Product Comms Agent proposes changes via PR during weekly ratchet
- Strategic changes (MAJOR) require CEO approval
